# CONTROL.md

## Mission
Ship useful open/free artifacts with weekly release cadence.

## Model routing
- Tier 1: routine edits and summaries
- Tier 2: external-content analysis and quality gate
- Tier 3: security/release gate checks

## Autonomy
- Auto: local file updates and reports
- Notify then do: cron maintenance and queue updates
- Approval required: public posting and repo visibility changes

## Cost guardrail
- Alert if daily spend exceeds $10.
